This section lists and introduces the blogs posted to the [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj):

### [Blog 1 - Secure Database Credentials for AWS Lambda with AWS Secrets Manager](3.1-Blog1/)
This blog introduces how to use AWS Secrets Manager to secure database credentials and pass them to AWS Lambda functions connecting to Amazon RDS (MySQL), eliminating hardcoding passwords in source code, and introducing automated periodic password rotation.

### [Blog 2 - Proactive Infrastructure Monitoring using Ambient Agents on Amazon Bedrock](3.2-Blog2/)
This blog introduces the AgentWatch solution, which provides a continuous Ambient Agent model based on Claude and LangChain on Amazon Bedrock, helping to automate infrastructure monitoring and optimize operational workflows with Human-in-the-Loop controls.

### [Blog 3 - Understanding VPC Links in Amazon API Gateway Private Integrations](3.3-Blog3/)
This blog digs into the core technologies behind VPC Link (such as AWS Hyperplane and AWS PrivateLink) and compares the fundamental architectural differences between VPC Links for REST APIs and HTTP APIs in API Gateway.
