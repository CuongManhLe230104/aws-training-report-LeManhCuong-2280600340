Tại đây là phần liệt kê, giới thiệu các blogs đã đăng trên [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj):

### [Blog 1 - Sử dụng AWS Secrets Manager bảo mật database credentials cho AWS Lambda](3.1-Blog1/)
Blog này giới thiệu cách sử dụng AWS Secrets Manager để bảo mật credentials của cơ sở dữ liệu và truyền chúng cho các hàm AWS Lambda khi kết nối với Amazon RDS (MySQL), loại bỏ việc hardcode mật khẩu trong mã nguồn và giới thiệu tính năng tự động xoay vòng mật khẩu (Rotation Schedule).

### [Blog 2 - Giám sát hạ tầng chủ động sử dụng Ambient Agents trên Amazon Bedrock](3.2-Blog2/)
Blog này giới thiệu giải pháp AgentWatch cung cấp một mô hình Ambient Agent (Đại lý môi trường) hoạt động liên tục dựa trên Claude và LangChain trên Amazon Bedrock, giúp giám sát hạ tầng tự động và tối ưu hóa quy trình vận hành thông qua quy trình phê duyệt Human-in-the-Loop.

### [Blog 3 - Tìm hiểu về VPC Link trong Private Integrations của Amazon API Gateway](3.3-Blog3/)
Blog này đi sâu vào các công nghệ cốt lõi đứng sau VPC Link (như AWS Hyperplane và AWS PrivateLink) và so sánh sự khác biệt cơ bản về mặt kiến trúc giữa VPC Link dành cho REST API và HTTP API trong API Gateway.
