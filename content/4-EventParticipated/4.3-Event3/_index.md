# Detailed Report: FCAJ Community Day - June 27, 2026

* **Event Name:** FCAJ Community Day - June 2026
* **Date & Time:** June 27, 2026
* **Location:** Online on youtube channel AWS Study Group
* **Role:** Attendee

---

### Description of Content and Main Activities

This online event hosted on the AWS Study Group YouTube channel provided a comprehensive look at cloud career progression, localizing Voice AI agents, DevOps automation, and secure AI assistant integrations. The main presentations included:

1. **Cloud/AI Career Path & Future (Speaker: Steve Tran - Founder of Cloud Thinker):**
   - Sharing his career journey from starting at age 19, working at Amazon, to founding Cloud Thinker.
   - Analyzing AI's impact on IT recruitment, highlighting the shift toward hiring senior profiles who effectively leverage AI.
   - Introducing Cloud Thinker's solutions for cloud infrastructure operations support.
2. **Voice AI Agent for the Vietnamese Market (Speakers: Nghi, Kiet, Trung - R AI):**
   - Showcasing Voice Agents applied in customer service call centers at major banks like VIB and VPBank.
   - Technical challenges in processing Vietnamese speech (accents, tones, interruption detection, regional dialects).
   - Designing localized AI architectures instead of relying on standard English speech-to-speech models.
3. **DevOps Agent - Automated Incident Troubleshooting (Speakers: Bao, Nguyen - Cloud Kinetics):**
   - Introducing DevOps AI Agents designed to automatically investigate system anomalies (e.g., website slowness from request spikes).
   - Proposing rapid automated mitigation workflows to minimize system downtime.
4. **AI Application (Amazon Q) in HR Management (Speakers: Truong, Minh Anh - Noventiq):**
   - Practical demonstration of using Amazon Q Business to automate traditional HR workflows.
   - Automatic data extraction, automated CV screening based on Job Description (JD) standards, candidate ranking, and automatic evaluation charts generation.
5. **Securing Amazon Q with MCP Server:**
   - Technical deep-dive on connecting Amazon Q Business securely to third-party tools (e.g., Zalo, Jira).
   - Implementing VPC networking and Model Context Protocol (MCP) proxy servers to prevent data exfiltration to the public cloud.

---

### Outcomes and Value Gained

* **Lessons Learned:** Understood how AI transforms recruitment standards and operational tasks (DevOps incident response and automated CV screening). Gained knowledge on how to securely bridge Amazon Q to external platforms without leaking private data to public clouds.
* **New Skills:** Learned localized voice processing constraints in AI and the implementation of MCP server gating rules.
* **Contribution to work:** The security and integration patterns (VPC and MCP) serve as a valuable reference for secure data isolation in the SaaS HR multi-tenant project.
