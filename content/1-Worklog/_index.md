**On this page**, you will find an overview of my weekly worklog during the Workforce Bootcamp at First Cloud AI Journey. This program was completed over 12 weeks, during which I transitioned from learning cloud essentials to building, deploying, securing, and optimizing a complex SaaS HR multi-tenant microservices application on AWS.

The weekly achievements and tasks are summarized below:

**Week 1:** [Getting familiar with AWS and basic AWS services](1.1-week1/) - Setting up the AWS Free Tier, configuring AWS CLI, launching EC2 instances, and configuring billing alerts.

**Week 2:** [AWS IAM & VPC Networking](1.2-week2/) - Establishing advanced IAM roles (Switch Role) and building a secure multi-subnet VPC network from scratch.

**Week 3:** [Microsoft Active Directory & VPC Peering](1.3-week3/) - Deploying Windows Active Directory (AD), RDGW, and establishing VPC Peering with cross-peer DNS resolution.

**Week 4:** [Network, Compute (LAMP) & RDS, S3](1.4-week4/) - Trienzlying a CRUD LAMP web application, Amazon RDS (MySQL) backups, and S3 secure bucket policies.

**Week 5:** [AWS CLI, Lightsail, CDN CloudFront & Docker](1.5-week5/) - Automating resources with CLI scripts, setting up CloudFront OAC, dockerizing applications, and using AWS Backup.

**Week 6:** [SaaS HR Microservices & Nginx API Gateway](1.6-week6/) - Designing and building SaaS HR microservices (Auth, Tenant, HR) and implementing Nginx gateway routing.

**Week 7:** [Serverless Architectures (Lambda, API Gateway, SAM, Cognito, Step Functions)](1.7-week7/) - Migrating the architecture to serverless, building with AWS SAM, and securing user pools with Amazon Cognito.

**Week 8:** [Infrastructure as Code (Terraform) & FinOps Cost Optimization](1.8-week8/) - Automating all cloud infrastructure via Terraform and executing FinOps strategies to cut operational costs by 61%.

**Week 9:** [Secrets Manager Database Rotation & VPC Link Analysis](1.9-week9/) - Implementing automated 30-day credential rotation with Secrets Manager and analyzing REST vs HTTP API Gateway VPC Links.

**Week 10:** [Static Frontend on S3 & Application Load Balancer (ALB) Routing](1.10-week10/) - Compiling and synchronizing React frontend to S3 with private access, and configuring ALB path-based routing.

**Week 11:** [Report structuring, worklog review & GitHub deployment](1.11-week11/) - Initializing the internship report structure, standardizing all weekly worklogs, writing technical blog posts on cloud architectures and FinOps, and deploying the Hugo site to GitHub Pages.

**Week 12:** [Internship report finalization & deliverables handover](1.12-week12/) - Finalizing the static report website, completing graduation internship reports (Báo cáo TTTN), weekly tracking forms, self-evaluation sheets, and executing final project handover.
